# finance-ai-learning

Mitt lärrepo för AI tooling, agentiska system och modern developer productivity — med fokus på tillämpningar inom ekonomi, controlling och CFO-arbete.

## Struktur

| Mapp | Innehåll |
|------|----------|
| `journal/` | Daterade läranteckningar — vad jag gjorde, insikter, nästa steg |
| `concepts/` | En fil per koncept i egna ord, med ekonomiexempel |
| `prompts/` | System prompts för controlleruppgifter |
| `projects/` | Praktiska projekt (varje projekt i egen mapp) |
| `templates/` | Mallar för journal, koncept och beslut |

Se även:
- `CLAUDE.md` — Instruktioner för Claude Code
- `decisions.md` — Arkitekturbeslut och motiveringar

## Lärandeplan

Följer instuderingsplanen med 7 moduler:

1. ~~Prompt Engineering & System Prompt Design~~ ← pågår
2. Tool Use & Function Calling via API
3. Claude Code — Praktisk AI-Assisterad Utveckling
4. MCP — Bygga & Ansluta Servrar
5. Skills & Cowork
6. Agentarkitektur & Orkestrering
7. Evals, Säkerhet & Produktion

## Komma igång

```bash
cd finance-ai-learning
# Starta Claude Code i projektet
claude
```
