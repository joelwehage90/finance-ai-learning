## Vad som fungerade

- Client Credentials-flödet (dec 2025) förenklade auth enormt — ingen refresh token behövs
- FastMCP gör det väldigt enkelt att exponera API:er som tools — bara dekorera funktioner
- `uv run --with-requirements` löser dependencies automatiskt utan manuell pip install
- Iterativt debuggande av seed-skriptet lärde mig mer om Fortnox API än dokumentationen

## Vad som inte fungerade / var förvirrande

- Fortnox har **inkonsistenta fältnamn** mellan kund- och leverantörsfakturor (InvoiceRows vs SupplierInvoiceRows, AccountNumber vs Account)
- Sandbox saknar räkenskapsår by default — måste skapas med exakt `AccountChartType: "Bas 2025"`
- Windows cp1252-encoding kraschar på unicode-symboler (✓/✗) — fick byta till ASCII
- Byte-literals (`b"..."`) i Python klarar inte svenska tecken (å/ä/ö) — måste använda `.encode("utf-8")`

## Insikter

- MCP är en superkraft för att ge Claude tillgång till externa system — och det var enklare att bygga än förväntat
